1. Clone repo and unzip.
2. Create local MySQL database, can use Laragon for this.
3. Copy and paste .env.example file, edit file and change database section.
4. Open in PHPStorm and in terminal window execute:

    ```sh
   composer install
    ```
    ```sh
    npm install
    ```
   ```sh
   npm run build
    ```
   ```sh
   php artisan migrate
    ```
